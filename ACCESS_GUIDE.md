## 🎬 完整演示流程

### 步骤1: 用户提交项目
1. 访问用户登录页
2. 使用 `testuser` / `test123` 登录
3. 在仪表盘点击"提交新项目"
4. 填写项目信息:
   - 企业名称: 例如"优质美妆店"
   - 商品品类: 选择"美妆"
   - 投流ROI: 输入 2.5
   - 退货率: 输入 20
   - 净利润率: 输入 25
   - 店铺评分: 输入 4.8
   - 运营时间: 输入 12
5. 点击"提交项目"
6. 记住返回的提交编号

### 步骤2: 管理员评分和审批
1. 访问管理员登录页
2. 使用 `admin` / `admin123` 登录
3. 在项目列表找到刚才提交的项目
4. 点击"查看详情"打开模态框
5. 点击"智能评分"按钮
6. 查看评分结果（应该是100分，通过）
7. 点击"审批通过"
8. 点击"上传协议"

### 步骤3: 用户查看结果
1. 切换回用户端
2. 在仪表盘查看项目列表

5. 可以看到完整的评分结果和工作流历史

## 💡 使用提示

1. **所有页面都支持直接访问** - 不需要从首页进入，可以直接打开任何页面
2. **登录状态会保存** - 登录后关闭浏览器再打开仍然是登录状态
3. **支持多个浏览器/标签页** - 可以同时打开用户端和管理员端进行对比
4. **刷新页面不会丢失状态** - 前端路由使用浏览器历史记录API
5. **点击浏览器后退按钮** - 可以正常返回上一页

---

## 🔧 如果页面显示空白

如果打开页面后看到空白，请尝试：

1. **硬刷新页面**: 按 `Ctrl+Shift+R` (Windows) 或 `Cmd+Shift+R` (Mac)
2. **清除浏览器缓存**: 
   - Chrome: 设置 → 隐私和安全 → 清除浏览数据
   - Firefox: 设置 → 隐私与安全 → 清除数据
3. **检查控制台错误**: 按 `F12` 打开开发者工具，查看 Console 标签
4. **使用隐身/无痕模式**: 新开一个无痕窗口访问
